Preface

Yiffy is a tool to help creators test their content against Gumroad's Iffy moderation tool ahead of time. It may not give the same result as Gumroad's internal checks.

Flagged accounts have their payments disabled without clear explanations of what triggered the flag or how to fix it.

Please read over disclaimer and usage before using this tool.

Tool

API Endpoint

API Key

Important: Do not share your API key with anyone.

Create an account and get an API key at Iffy. You can see how this tool is used for more information.

Text

Enter the text you want to moderate.

Images

Enter the URLs of images you want to moderate. Note: Iffy has a image size limit.

Disclaimer

Use due diligence when using any tool. This tool is provided as-is and does not guarantee that your content will not be flagged or removed by any platform. DO NOT share or expose your API key to anyone.

Usage

  • Follow the Iffy setup and acquire an API key: https://iffy.com/docs/quickstart (Note: that API key will not be free).
  • Input the API key in the API Key field.
  • Input either text content or image URLs for moderation.
  • Click the Moderate button to send the request.
  • Visit the dashboard on Iffy for a more detailed breakdown of the results.
